Detailed Description

template<class VertexProperty, connectedness Representation, directional Directed>
requires (!std::is_same_v<VertexProperty, no_property>)
class quetzal::geography::graph< VertexProperty, no_property, Representation, Directed >

A graph class with information attached to vertices.

Template Parameters
VertexPropertyThe type of information to store at each vertex.

Vertices embed a user-defined type of information. Edges embed no additional information. This graph structure is bidirected and allows to model a forest of disconnected graphs and isolated vertices.

Invariant
Guarantees that if \((u,v)\) is an edge of the graph, then \((v,u)\) is also an edge.
